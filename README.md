# NEU Library Visitor Log System

This is a React-based application designed to track and manage visitor entries at the NEU Library. It uses Firebase for real-time data storage and Google Authentication.

### 📁 GitHub Repository
[https://github.com/GinalineLopez/neu-library-system](https://github.com/GinalineLopez/neu-library-system)

### Key Features:
- **Google Authentication:** Secure login for students and employees.
- **Visitor Terminal:** Easy entry log for purpose of visit and college.
- **Admin Dashboard:** Real-time analytics and visitor tracking.
- **PDF Reports:** Downloadable visitor logs for documentation.
- **Custom UI:** Personalized background and professional layout.

### Technologies Used:
- React.js
- Firebase (Auth & Firestore)
- Recharts (Analytics)
- jsPDF (Report generation)
- Vercel (Hosting)
